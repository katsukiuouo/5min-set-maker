プライバシーポリシー

最終更新日：2026年7月2日

1. 取得する情報

本サービスでは、氏名、メールアドレスなどの個人情報を取得しません。

2. ブラウザへの保存について

本サービスでは、動画リストや設定、生成した10分セットをブラウザのLocalStorageへ保存します。

これらの情報は利用者の端末内にのみ保存され、運営者へ送信されることはありません。

3. 共有機能について

利用者が「Xで共有」機能を利用した場合、共有ページを作成するため、以下の情報をSupabaseへ保存します。

  • 動画ID
  • 動画タイトル
  • チャンネル名
  • 再生時間
  • サムネイルURL

保存されるデータには、利用者を特定できる個人情報は含まれません。

4. YouTube Data APIの利用

本サービスでは、YouTube Data API Servicesを利用して動画情報を取得しています。

取得した情報は、本サービスの機能を提供する目的にのみ利用します。

5. 利用する外部サービス

本サービスでは、以下の外部サービスを利用しています。

  • YouTube Data API Services
  • Supabase
  • Vercel
  • X（共有機能）

各サービスの利用については、それぞれの利用規約およびプライバシーポリシーが適用されます。

6. ポリシーの変更

本ポリシーは、必要に応じて変更する場合があります。
